# Neobrutalist Design Identity

The visual language is deliberately neobrutalist: zero border-radius on all elements, thick 3px solid borders, offset box-shadows (5px/8px/12px), and high-contrast accent colors (red, blue, pink) on neutral surfaces. This identity is baked into ~200+ CSS selectors via tokens (`--border-default`, `--shadow-sm/md/lg`) and cannot be swapped for a different aesthetic without rewriting the entire stylesheet.

This is the portfolio's differentiator. Generic portfolios use rounded corners and subtle gradients. Neobrutalism signals boldness and confidence — qualities the site attributes to its author.

Alternatives considered: Minimalism (too safe, forgettable), glassmorphism (fragile across browsers), retro-futurism (too niche). Neobrutalism balances memorability with readability.
