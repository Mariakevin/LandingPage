# Portfolio — Neobrutalist Dark Mode

Personal portfolio website built as a single HTML file with neobrutalist design language.

## Features

- Fully dark mode
- Custom animated cursor with hover states
- Multi-stage page loader
- Scroll-driven reveal animations
- Stats count-up on scroll
- Skill proficiency bars with IntersectionObserver animation
- Project filter by category
- Stacking toast notifications
- Contact form with inline validation + draft save
- Education timeline
- Client testimonials with avatars
- Mobile slide-out nav overlay
- Keyboard shortcuts (`S` Skills, `A` About, `P` Projects, `C` Contact, `?` help)
- Focus trap in dialogs (accessible)
- Back to top button
- Konami code easter egg
- `prefers-reduced-motion` support

## Tech

- Pure HTML + CSS + JavaScript (zero dependencies)
- Space Grotesk + Space Mono fonts (Google Fonts)
- SVG icons for social links
- CSS `animation-timeline` scroll reveals with IntersectionObserver fallback

## Deploy

Push to GitHub and enable GitHub Pages (Settings → Pages → branch `main`, root folder). Or drag `index.html` to [Netlify Drop](https://app.netlify.com/drop).
